
# Django Submit Project

## Description
A Django-based CRUD application with a landing page, create, update, and error handling.

## Features
- Home page
- Create record
- Update record
- Static files handling
- Django ORM & MVT structure

## How to Run Locally
1. Clone the repository
2. Create virtual environment
3. Install requirements
4. Run server

## Tech Stack
- Python
- Django
- HTML, CSS
